---
name: i18n URL-based routing
description: All routes prefixed with /:lang, useLocalizedNavigate for navigation, LanguageRouter syncs i18n
type: feature
---
- 7 languages: en, de, es, fr, nl, ja, ko
- All routes wrapped in `/:lang` prefix in App.tsx via LanguageRouter component
- `useLocalizedNavigate()` hook auto-prefixes paths with current lang
- `useLocalizedPath()` hook for `<Link to={lp("/path")}>` pattern
- LanguageSwitcher navigates to new lang prefix instead of just calling changeLanguage
- SEO: hreflang tags added dynamically, html lang attribute set
- Domain: ordex-systems.com/{lang}/
- i18n detection order: path → localStorage → navigator
